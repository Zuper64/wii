cIOS

Installing custom IOSs allows the Wii to be able to run more homebrew, such as game backup loaders.

  For this guide, you will need:  

  • A Wii with the homebrew channel  

  • An SD card formatted to FAT32  

  • d2x cIOS installer  

  • NUS Downloader (Offline Only)  

Offline installation

You may skip this section if your Wii has internet access.

1. Download NUS downloader from here and extract it with 7zip.

2. Locate "NUS Downloader.exe" and double click it to run it.

3. Select "Database" then "IOS".

4. Select IOS38, then v4123.

5. Ensure that "Pack WAD" is checked, then select "Start NUS Download".

6. Repeat steps 4 and 5 for IOS56 v5661, IOS57 v5918 and IOS58 v6175.

7. There should now be a "titles" folder in the same directory as NUS Downloader. Locate the wad files you downloaded and place them in the root of your SD card.

Installing cIOS

Continuation of offline guide/ Start of online guide.

1. Download d2x cIOS Installer from here and extract it to your SD card.

2. Insert your SD card into your Wii and go to the homebrew channel.

3. Launch d2x cIOS Installer.

4. Press A to continue and set it to "Select cIOS: d2x-v11-beta1, Select cIOS base: 38, Select cIOS slot: 248, Select cIOS version: 65535"

5. Press A twice to install. Then, press A again to return.

6. Set the installation to "Select cIOS: d2x-v11-beta1, Select cIOS base: 56, Select cIOS slot: 249, Select cIOS version: 65535"

7. Press A twice to install. Then, press A again to return.

8. Set the installation to "Select cIOS: d2x-v11-beta1, Select cIOS base: 57, Select cIOS slot: 250, Select cIOS version: 65535"

9. Press A twice to install. Then, press A again to return.

10. Set the installation to "Select cIOS: d2x-v11-beta1, Select cIOS base: 58, Select cIOS slot: 251, Select cIOS version: 65535"

11. Press A twice to install. Then, press B to exit to the homebrew channel.
